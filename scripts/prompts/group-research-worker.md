## 籌碼與風險標記判讀

task 的 `members` 可能帶有以下額外欄位，有則判讀，無則忽略：

- `chips`：三大法人買賣超（單位：張），含 `foreignNet`（外資）、`trustNet`（投信）、`dealerNet`（自營）。
  - 投信與外資同方向買超的成員是族群可信度加分，可在 `stockNotes` 或 story 點出。
  - 若任一成員外資或投信有明顯賣超（負值），視為信心折扣，`confidence` 不宜給 `high`。
- `dayTradeRatio`：當日當沖成交量佔比（%）。
  - 高於 40% 代表隔日沖盤主導，延續性評價降低，story 可點出「當沖比重高」。
- `flags`：
  - `attention: true`：注意股，流動性或財務面有疑慮；不可作為族群核心論據，story 若提及此股需點出其投機屬性。
  - `disposition: true`：處置股，交易受限；同上，需特別說明。
  - `lowLiquidity: true`：成交金額低於 5000 萬，族群代表性低；不宜放入 `stockNotes` 重點個股。
- `overnightDump: true`：該成員昨日漲停、今日當沖率高且收黑，符合隔日沖出貨足跡（昨漲停今爆當沖收黑）。分析時必須明確點出此股為投機籌碼、隔日沖主導，**不可作為族群基本面核心或代表股**，story 與 stockNotes 需與族群題材區隔說明。

輸出檔格式固定為：
```json
{
  "category": "...",
  "direction": "gainer",
  "story": "...",
  "confidence": "high",
  "keyDrivers": ["...", "..."],
  "stockNotes": [
    {"stock":"南亞科(2408)","note":"..."}
  ]
}
```

規則：
- `confidence` 只能是 `high` / `medium` / `low`
- 如果這個族群沒有明顯共通新聞，仍要用產業知識寫完整題材：需求端、供給端、報價、庫存、技術升級、政策、法人資金或代表股事件至少選 2 個角度。
- 優先抓 1-2 個最強催化，不要為了補滿資訊而過度延長搜尋。
- 若 3 天內缺乏整齊共通新聞，請盡快輸出 `low` 或 `medium` 信心結果，但 story 仍要像完整分析，不可只寫「沒有新聞」。
- 下跌族群的 story 也要有內容：說明是報價/庫存/需求疑慮、前波漲多後籌碼調節、或題材落空；同時交代該族群原本的市場亮點，讓讀者知道為什麼它值得被放在一起看。
