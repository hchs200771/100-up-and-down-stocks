# MM 讀數據的方法

來源：EP189（2026-03 美伊戰爭油價）、EP190（2026-03 油價破百）、EP187（2026-02 黃金 vs 法幣）
逐字稿，主講 Jason / Ralice。**新舊衝突以較新集數為準**。

## 通用心法（讀任何數據前先問這四題）
1. **供給面 還是 需求面？** 衝擊來自供給中斷（地緣、減產）還是需求變化？決定持續性。
2. **消息面 還是 基本面？** 是情緒性波動，還是真的改變供需/獲利的結構？（呼應主框架第 0 步）
3. **實體規模多大？** 不要只看新聞情緒，**把實際的量算出來**（見原油例：運量、缺口佔比）。
4. **設門檻 + 配追蹤指標**：把關鍵變數分檔（情境階梯），並列出後續用哪些數據驗證。

每集開場的「**行情回顧**」固定動作：
本週 美股/亞股/台股 表現（創高/回調）→ 找出**最大變動因子** → 看**避險資產**（黃金、美元、美債殖利率）怎麼反應 → 判斷市場在定價什麼。

## 原油 Playbook（最完整的範例）

**第 1 步：量實體供給衝擊規模（MM 的招牌——先算量再談價）**
- 荷姆茲海峽正常每天約 100–300 艘油輪、運量約 **2000 萬桶**原油+成品油；中斷後流量降 **90–100%**。
- 佔全球供給比重：荷姆茲 ≈ 全球 **10%** 原油供給。
- 緊張度旁證：租油輪費用（戰時每天 40 萬美元 = 20 年最高）。
- MM 會用網站的荷姆茲海峽**運量/船舶數圖表**（甚至放 GIF 動態）直接看通行狀況。

**第 2 步：供給 vs 需求、有無緩衝**
- 用「**全球石油供給 − 需求**」看有沒有緩衝空間。
- OPEC+ 動向：增產/減產、**剩餘產能能否補缺口**。關鍵判斷：若 **8–10% 供給被切斷**，沒有產油國有足夠剩餘產能補足 → 油價會「不理性上漲」。
- SPR（戰略儲備）釋放：歷史上對油價緩解**只是短期**（每天抵消約 200–220 萬桶，相對 2000 萬桶運量杯水車薪）。

**第 3 步：油價門檻階梯 → 通膨 → Fed（接主框架）**

| WTI 油價 | 性質 | 年底核心通膨 / Fed |
|---|---|---|
| ~$60 | 回歸基本面 | 通膨回落、降息空間最大 |
| ~$75 | 供給側衝擊 | 通膨偏高、降息空間壓縮 |
| ~$90 | 供給側衝擊（高檔） | 通膨卡住、降息空間更小 |
| **≥$100** | **臨界門檻** | 長時間維持才會**衝擊需求面**、通膨預期恐失控、降息無望甚至升息 |

- **關鍵 know-how**：**$100 以下多屬「供給側衝擊」，需求動能還沒受影響**；唯有**長時間 $100 以上**（如 2022 俄烏）才會傳導到需求面、估值面臨戴維斯雙殺。

**第 4 步：歷史類比 + 差異點**
- 類比 2022 俄烏（油價長期破百 → 衝擊需求、暴力升息）。
- **本次差異**：① AI 需求全面爆發，企業上中下游**庫存在低檔**（去庫存壓力小），與 2022 庫存堆積相反；
  ② 用**實質利率 vs 中性利率**判斷政策限制性，聯準會面對供給衝擊的應對空間不同。

**第 5 步：追蹤指標**：荷姆茲運量/船舶數、EIA 原油庫存、OPEC+ 會議、SPR 釋放、油價是否壓到終端需求、CPI/PPI 中能源項。

## 黃金 / 貴金屬 Playbook

**基本面＝三股力量**（判斷多頭是否成立看這三個）：
1. **地緣週期**：避險需求 + 各國央行囤金（累積黃金儲備）。
2. **貨幣政策 / 實質利率**：Fed 停止升息或降息 → 降低持有黃金的**機會成本**（黃金不孳息）→ 利多。
3. **債務週期 / 法幣貶值**：美債 38 兆、利息成本創高、法幣購買力被稀釋 → 買黃金避險。
   長期看 **CPI / 美元購買力 vs 金價**：法幣貶值趨勢「幾乎無解」，金價長線上行反映此。

**籌碼面 vs 基本面（重要 know-how）**：
- 黃金/白銀**單日暴跌**（白銀 −38%、黃金 −13%）常是**籌碼堆疊**所致——期貨轉倉日、選擇權造市商避險平倉、保證金追繳——**不改變基本面論述**。
- 解讀暴跌時先分清「**籌碼洗盤**」還是「**基本面反轉**」；若基本面三力量未變，暴跌多為錯殺。

**配置思維**：黃金是投資組合的**系統性避險**（對沖法幣貶值/債務違約），強調**配置比重**而非槓桿操作；白銀兼具避險+工業（AI 需求）雙重屬性，波動更大。

**後美元時代框架**（黃金的長線戰略價值）：特里芬難題、石油美元+債務美元閉環、去美元化（歐元超主權實驗、去中心化貨幣、人民幣）。三種情境（美元續命 / 多邊貨幣體系）**殊途同歸 → 黃金價值都存在**。

## 輸出模板（套到一個新數據/原物料）
```markdown
## 數據：<名稱與當前數值>
- 供給/需求？消息/基本面？
- 實體規模：<把量算出來，不只看情緒>
- 門檻階梯：<關鍵價位/數值 → 後續變數（通膨/Fed/需求）>
- 歷史類比：類似 <事件>，差異：1)… 2)…
- 籌碼 vs 基本面：<這波波動是哪一種>
- 追蹤指標：<MM 網站哪些圖表/數據>
- 對資產：油價/金價 → 通膨 → 殖利率/美元/股市估值 的方向
```

## 演進註記（「新者優先」）
- 2026 起新增變數：**AI 需求**改變了原物料的庫存循環判讀（去庫存壓力小於 2022）；
  黃金多頭的「債務週期/法幣貶值」力量因美債規模續創高而強化。解讀新數據時以此為準。
