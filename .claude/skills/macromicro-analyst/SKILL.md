# MacroMicro 研究員分析框架

代理財經 M 平方研究員（Rachel、Ryan、Vivianna、Jat、Ralice、Jason、Dylan…）的思考骨架，
對新事件做結構化判讀。**核心三層分析經 EP200 逐字稿驗證；部分輔助步驟仍為 show notes 推論。
輸出為研究分析，不構成投資建議。**

## Skill 組
- **本檔 [[macromicro-analyst]]**：主框架（任何事件的通用拆解）。
- **[[macromicro-fed-reading]]**：Fed/FOMC 解讀（點陣圖、聲明稿、SEP、流動性）。
- **[[macromicro-data-reading]]**：數據/原物料解讀（原油、黃金、行情回顧）。
- **[[researchers]]**：各研究員視角與風格，決定用誰的角度切入。

## 何時用

使用者丟出一個經濟/市場事件並想知道「影響是什麼、機率多高、對股市衝擊多大」時。
典型觸發：地緣衝突、Fed/日央/各國央行升降息、CPI/就業數據、關稅、重要財報、產業轉折。
依事件領域可轉用 [[macromicro-fed-reading]] / [[macromicro-data-reading]]，並參照 [[researchers]] 選視角。

## 核心心法（MM 的招牌套路，照順序跑）

### 第 0 步｜事件定性（最關鍵的第一刀）
先問一句話：**「這件事是『消息面』雜訊，還是會傷到『基本面』？」**
MM 研究員反覆強調：市場短期被消息面驅動，但中期定價回到基本面。
- 只影響情緒/估值（消息面）→ 多半是「錯殺」或「超漲」，會均值回歸。
- 真的改變企業獲利或景氣循環方向（基本面）→ 才是趨勢級衝擊。
判斷事件落點，決定後面要不要當大事看。

### 第 1 步｜MM 三層分析（主框架，Rachel 於 EP200 [07:56]–[14:53] 親述）

**這是一套三步驟的研究流程，不是三個分析面向。** 一句話：
**先窮舉「會怎樣」（情境）→ 再用邏輯壓「最可能怎樣」（勝率）→ 最後用數據盯「真的變這樣了嗎」（確認）。**
三層剛好對應你要的產出：影響鏈 → 機率 → 持續驗證。

---

#### 第一層 ─ 情境拆解：把事件變成一張「變數 → 後果」對照表 `[07:58]`

**要做的事**：找出**驅動這個事件的關鍵變數**，沿著它拉一條**情境階梯**，
每一格往下推到底（變數值 → 中介後果 → 政策反應 → 市場反應）。
重點是「事前」把每一格的 if-then 算好，等現實落到哪一格，你立刻知道它意味著什麼。
Rachel 稱這是「研究員必須具備的基本功力」。

**怎麼挑關鍵變數**：挑那條**正在運作的傳導鏈的源頭**（見第 2 步）。
美伊戰爭的關鍵變數不是「會不會打」，而是**油價**——因為
`油價 → 通膨 → Fed 利率 → 股市估值` 才是真正影響股市的鏈，「打不打」只有透過油價才會傳到股價。

**EP200 油價情境階梯**（研究員親算，瑞海/Ryan 報告）：

| WTI 油價 | → 年底核心通膨 | → Fed 政策空間 | → 對股市估值 |
|---|---|---|---|
| ~$60 | 3.2% 回落到 2.4% | 還有 **2–3 碼**降息空間 | 利多（資金成本降） |
| ~$75 | 偏高、回落變慢 | 壓縮到 **1–2 碼** | 中性偏空 |
| ~$90 | 卡住不再降 | **失去降息空間** | 利空 |
| ≥$100 | 續揚 | 反轉成**升息壓力** | 重大利空（戴維斯雙殺風險） |

讀法：油價是因、通膨是果、Fed 是反應、估值是終點。把同一條鏈在不同油價下的結局排清楚，
第一層就完成。**這層只窮舉可能、先不下注。**

---

#### 第二層 ─ 判斷勝率：用邏輯壓「哪一格最可能」 `[09:26][12:29]`

**要做的事**：不要平均看待所有情境，用邏輯推出**哪一格最可能發生**、給出機率傾向。
MM 用三個具體工具：

**(1) 紅線分析（最核心）** — 找出關鍵決策者「被迫出手的臨界值」，用來預測他的行為。
EP200 例：川普的兩條紅線——
- **紅線①：10 年期美債殖利率 4.4–4.5%。** 川普要低利率，一破 4.5% 他壓力就大。
  *實證規律*：殖利率每逼近/破 4.5% → 川普就放軟。對照他上任後每一次：
  破線 → 對等關稅暫緩 90 天 → 再破線 → 陸續宣布與各國貿易協議 → 4 月初再破 →
  釋放對伊朗談判訊號 → 近期又破 → 白宮喊話「談判算順利」。**「踩線就退」反覆出現，故可預測。**
- **紅線②：WTI 油價 ~$100。** 過高 → 通膨 → 利率↑、且壓到需求，川普同樣緊盯。

  推論：既然他有紅線、且每次踩到就退，就能判斷**「最壞狀況很可能已發生在當下」**——
  踩線那一刻往往就是頂，之後他會自己把它壓回去。

**(2) 歷史類比** — 把過去類似事件「已驗證的邏輯」整套搬過來。
EP200 例：用去年關稅戰的結論類比美伊戰爭。當時判定「關稅最壞已過」靠的是**關稅三大矛盾**——
① 雙赤字不可能同時下降 ② 進口物價大漲反而拖垮經濟 ③ 美國債務不可持續；
所以關稅注定只會放緩。美伊戰爭套同一套「川普踩到紅線就收手」的邏輯，得到同樣的「最壞已過」。

**(3) 痛苦指數（把紅線量化成一個儀表）** — Jason 做的「川普痛苦指數」，
把所有會逼川普讓步的壓力來源**合成一個可追蹤的數字**，成分：
① 10 年期公債殖利率 ② 川普施政滿意度 ③ 年底期中選舉壓力 ④ WTI 油價 ⑤ EIA 原油庫存。
*邏輯*：指數越高 = 川普越痛 = 越可能放軟 = 「最壞已過」這個情境的勝率越高。
它等於「多條紅線的綜合溫度計」，把第二層從定性判斷變成能逐週追蹤的指標。

**第二層產出**：在第一層那張階梯上標機率——例如「$90–100 區間震盪、川普踩線壓回」勝率最高，
「失控破百續揚」勝率低。

---

#### 第三層 ─ 數據確認：用後續數據盯「真的傷到基本面了嗎」 `[12:38]`

**要做的事**：判斷不是一次性的，用陸續公布的數據**持續驗證**第二層的押注，
核心是回答第 0 步那題——**這次衝擊到底是「消息面」還是真的「傷到基本面」？**

**關鍵判別法（重要 domain know-how）**：看漲價/衝擊是**全面擴散**還是**集中且一次性**。
- 集中在能源、一次性項目 → 屬消息面 → 會過去 → 維持「最壞已過」。
- 全面擴散到核心項目 → 傷基本面 → 升級成趨勢級利空 → 推翻原判斷。

**EP200 實例**：3 月 CPI/PPI 雖高於預期，但拆開看——
- CPI 攀升**主要是租金的一次性調整**；
- PPI 攀升**主要是運輸、貿易等跟油價相關的服務項**。

  → 結論：漲價仍**集中在能源項、未全面擴散** → 還沒傷到基本面 → 支持「最壞已過、可續抱」。

**持續盯的數據**（用來動態回頭修正第二層勝率）：每次 CPI/PPI 公布時的市場反應、
EIA 原油庫存（庫存掉 → 油價壓力 → 痛苦指數升）、油價是否真的壓到終端需求。

---

#### 三層合起來怎麼跑（EP200 完整走一遍）

1. **情境拆解**：油價 $60/$75/$90/$100 各對應 Fed 降息空間與估值結局（上面那張階梯）。
2. **判斷勝率**：川普兩條紅線 + 關稅戰類比 + 川普痛苦指數 → 判斷「3 月美伊戰爭最壞狀況已過」，
   基準情境＝「油價 $90–100 區間、川普踩線就壓回」。
3. **數據確認**：CPI 是租金一次性、PPI 是油價相關服務 → 漲價未全面擴散 →
   維持「最壞已過、美股/台股可續抱」，並持續盯 CPI/PPI 與原油庫存。

### 第 2 步｜傳導鏈（建情境用的骨幹，餵給第一層）
畫出 **事件 → 中介變數 → 終端資產** 的鏈條，逐節點標方向：

```
地緣/供給衝擊 → 油價/原物料 → 通膨 → 央行利率預期 → 美債殖利率 → 股市估值 + 匯率
政策/央行    → 利率/流動性 → 殖利率 + 美元 → 成長股估值 + 新興市場 + 黃金
產業趨勢     → 資本支出/獲利 → 類股輪動 → 指數權值
```
每一節點問：**這條鏈會不會斷？**（MM 常說「原油缺口衝擊其實不大」「荷姆茲海峽是否真的斷運」——
驗證實體衝擊規模，而非只看消息。）

### 第 3 步｜定價階段判斷：殺估值 vs 殺基本面（Dylan 專長，EP196/193/192）
給衝擊評分前，先用 MM 的量化工具定位「市場現在殺的是什麼、跌得合不合理」：

**(1) 殺估值 vs 殺基本面（最核心的定價判斷）**
- **殺估值**：利率上漲 → 估值(PE)下修；跌幅通常**較小**，常是**錯殺**、可低接。
- **殺基本面（殺 EPS）**：需求被修正 → EPS 下修；**傷筋動骨**、跌幅大。
- **戴維斯雙殺**：估值與 EPS **同時下修**（最壞情境）。
- 判別線：例 油價長期 $100+ → 從「殺估值」升級到「殺需求/EPS」。**先確認現在殺的是哪一種，再決定要不要怕。**

**(2) MM 自編指數（交叉驗證，避免單一指標誤判）**
- **基本面指數**：股 / 匯 / 債 / 原物料各一條，以 **0** 為界，>0 = 基本面支撐。
  結合供需計算（例：原油現價高、但基本面指數偏低 → 價格是「短期外生變數」、非基本面驅動）。含**經濟預期指數 / 衰退機率**。
- **MM 牛熊指數**：0–100 綜合分數、動態更新、有歷史回測；可看全球或分國家。
  由基本面（製造業、就業）＋估值組成。用途：判斷**行情見頂/落底了沒**（中東戰爭前牛熊指標已見頂趨緩 → 行情確實見頂）。
- **估值監控**：巴菲特指標等；「爆表」= 過熱警訊，回中性 = 不過熱。
- 「**七大核心指數**」用來交叉驗證；多數指標同向才高信心。

**(3) 找錯殺**：拉開 11 個類股板塊看輪動（殺估值時科技股不一定殺最兇）；
到 MM 網站找「不該這麼差、被錯殺」的資產 = 低接標的。

### 第 4 步｜對股市衝擊評分
對每個關鍵市場（台股 / 美股 / 各別類股 / 美債 / 美元 / 黃金）給一格：

- **方向**：利多 / 利空 / 中性
- **強度**：🔴重大（趨勢級，動基本面 / 殺 EPS）/ 🟡中等（修估值/數週）/ 🟢輕微（消息面雜訊）
- **時間**：短期（數日-數週）/ 中期（數月）
- **錯殺/受惠標的**：被超賣可低接的、真受惠的

用第 3 步的 **基本面指數 / 牛熊指數 / 七大核心指數**交叉驗證方向（多數指標同向才高信心）。

## 事件類型 Playbook

針對使用者明確提到的兩類，給專屬重點（其餘事件回到核心心法）：

### A. 地緣危機（戰爭/制裁/海峽封鎖）
1. **先量實體衝擊規模**：真正中斷的供給/運量有多大？（MM：缺口常被高估）
2. 主傳導：地緣 → 油價/原物料 → 通膨 → Fed 降息空間縮 → 殖利率↑ → 成長股估值壓力。
3. 避險資產：黃金、美元、美債通常受惠，但要看 Fed 起點。
4. 定價階段：市場是否已從「最壞情境」轉向「基本面再評估」？（常見利空鈍化 → 反彈）
5. 類比：俄烏戰爭（暴力升息）vs 本次的差異。
6. 多數歷史地緣事件對股市是**短期利空、中期錯殺反彈**，除非實體傷及景氣。

### B. 央行利率（Fed / 日央 / 各國）
1. 讀三件套：**聲明稿變化、點陣圖、SEP（經濟預測）**。
2. 鷹/鴿相對「市場已 Price in 的預期」——意外才有行情（預期差）。
3. 資金面：是否搭配流動性操作（購債/QT/QE）？影響短端 vs 長端。
4. 傳導：利率預期 → 美債殖利率 → 美元 → 成長股估值 + 新興市場資金流 + 黃金。
5. **日央特例（Vivianna，EP182/201 驗證）**：
   - 日圓資產命運掌握在**首相 vs 日央行長**手裡（高市＝安倍路線/財政刺激，與日央升息方向衝突）→ MM 稱「日本首相與央行行長對決」。
   - 升息 → 日圓升 → 套利交易（carry trade）平倉 → 全球風險資產去槓桿。
   - **但要看升息的「機率與步調」**：財政刺激 + 債務壓力會讓日央**升息更慢**；MM 判斷「貨幣正常化的黃金窗口正在關閉」。
   - 現況通常是**股強、債弱、匯貶（或止穩）**；日股因升息慢 + 財政刺激「不會太差」。看美元兌日圓位階（如 156 = 偏弱）。
6. 對台股：間接——透過美元/殖利率/外資流動與科技股估值傳導。**台美股配置看製造業週期**（上行加碼台股、下行加碼美股，見 [[researchers]] Vivianna）。

## 限制與誠實聲明

  沿用其公開用語，但具體計算未知。

- 不是投資建議。
